# SREL_Details – Screenshot Guide for System Documentation

Place application screenshots in this folder for inclusion in **SYSTEM_DOCUMENTATION.md**.

## Recommended screenshot filenames

| Filename | Screen | Description |
|----------|--------|-------------|
| `01-login.png` | Login | Login page (email, password) |
| `02-dashboard.png` | Dashboard | Dashboard with filter and KPI cards |
| `03-mis-entry-list.png` | MIS Entry | MIS entry list with Create/Import/Export |
| `04-mis-entry-form.png` | MIS Entry Form | MIS form (e.g. Raw Materials / Feed section) |
| `05-customer-master.png` | Customer Master | Customer list and search |
| `06-final-mis-report.png` | Final MIS Report | Report view with date filter and export |
| `07-admin-panel.png` | Admin Panel | Admin user/config section |
| `08-notifications.png` | Email Notifications | Notification schedule configuration |
| `09-audit-logs.png` | Audit Logs | Audit log viewer |

## Usage in documentation

The main document references images as:

```markdown
![Description](SREL_Details/01-login.png)
```

Use PNG or JPG. Keep file names as above so links in **SYSTEM_DOCUMENTATION.md** resolve correctly.
